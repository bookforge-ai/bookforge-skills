## What Are Agent Skills?

Agent skills are `SKILL.md` files that give AI coding agents expert-level knowledge for specific tasks. When you install a skill, your agent gains structured decision frameworks, checklists, and workflows drawn from authoritative books — without you having to read them first.

Skills follow the open [Agent Skills specification](https://agentskills.io/specification) and work across all major agent platforms.

## Installation

### Claude Code

```bash
claude skills add bookforge-ai/bookforge-skills
```

### Codex CLI

```bash
codex skills add bookforge-ai/bookforge-skills
```

### Gemini CLI

```bash
gemini skills install https://github.com/bookforge-ai/bookforge-skills.git
```

### Manual (any platform)

```bash
git clone https://github.com/bookforge-ai/bookforge-skills.git
# Copy desired skill directories from skills/ into your agent's skills folder
```

## How Skills Are Made

Skills are generated by the [BookForge pipeline](https://github.com/bookforge-ai/bookforge), which extracts actionable knowledge from non-fiction books and synthesizes it into agent-executable skill files. Each skill is tested against a baseline to verify it adds genuine value.
